# Problema del modelo tecnologico 
## Software dependiente de subscripciones por servicio
---
Antes tú comprabas el servicio de un software de manera indefinida
Ahora debes estar pagando todos los meses por ese mismo servicio

# Solucion Kike
Usar programas gratuitos o de pago único en lugar de suscripciones mensuales.
Así el usuario paga solo una vez o no paga nada y puede seguir usando el programa
sin depender de pagos mensuales ni perder el acceso si deja de pagar.
